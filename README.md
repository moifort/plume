# Plume

Light dependency injection for java

## Maven

```xml
<properties>
        <plume.version>1.8</plume.version>
</properties>

<dependencies>
    <dependency>
        <groupId>com.github.moifort</groupId>
        <artifactId>plume</artifactId>
        <version>${plume.version}</version>
    </dependency>
<dependencies>
```
